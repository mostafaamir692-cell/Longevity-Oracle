Apply the following typography system to this website. Do not change any layouts, colors, images, or functionality — only update text styling.

Fonts: Import from Google Fonts: Inter (weights 300, 400, 500, 600) for body text, and Playfair Display (weights 400–700, including italic 400) for headings. Set font-sans to Inter and font-display to Playfair Display.

Base rules:

All h1–h6: font-display, tracking-tight, text-foreground, text-wrap: balance
All p: text-wrap: pretty
Body: font-sans, antialiased, font-feature-settings: "kern" 1, "liga" 1, "calt" 1
Heading sizes:

Hero h1: text-5xl md:text-7xl lg:text-8xl font-display font-bold leading-tight
Section h2: text-3xl md:text-5xl font-display font-bold leading-tight
Card h3 (large): text-2xl md:text-3xl font-display font-bold leading-tight
Card h3 (small): text-lg font-bold leading-tight

Body text hierarchy:

Primary body: text-foreground text-base leading-relaxed
Secondary/supporting: text-foreground/60 text-sm leading-relaxed
Dimmed/subtle: text-foreground/35 text-sm font-light or text-muted-foreground text-sm

Micro-labels (used above data points, durations, categories):

text-[10px] font-semibold uppercase tracking-widest text-foreground/40
